# Pandas Exercises

Data manipulation and analysis exercises using Pandas library.

## Setup

1. Create virtual environment:
```bash
python3 -m venv ex00
source ex00/bin/activate  # On Windows: ex00\Scripts\activate
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Launch Jupyter:
```bash
jupyter notebook
```

## Exercises

- **Exercise 1:** DataFrame creation from NumPy arrays and Pandas Series
- **Exercise 2:** Electric power consumption data analysis
- **Exercise 3:** E-commerce purchases analysis
- **Exercise 4:** Handling missing values

## Files

- `pandas.ipynb` - Main notebook with all exercises
- `playground.py` - Testing ground for code before adding to notebook
- `INSTRUCTIONS.md` - Detailed exercise instructions
